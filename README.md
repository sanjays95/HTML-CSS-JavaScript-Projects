Projects developed during internship and online bootcamp courses.

1. **Wirecar Project** - The simple application aims to calculate the CVV based on VISA standards. It takes the PAN, expiry date and service code. It uses a 64bit DES encryption alogrithm. (http://www.tero.co.uk/des/code.php). Still in refactoring stage
2. **Patatap Project** - Create melodious tone using keyboard strokes. Based on Paper.js
3. **Colour Guess Game** - A smiple browser game where you have to select the colour based on the given RGB value.
4. **Odd-One-Out** - Browser Game to find out the odd smileys.
5 **To-DO List** - Simplest of all todo list using jquery.
